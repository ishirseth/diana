# CAMS – Cat Adoption Management System (Project Page)

This repository contains a simple webpage created as part of my capstone project, CAMS (Cat Adoption Management System).

## About the project

CAMS is a mobile-first platform designed to support animal rescue organizations by improving and structuring the adoption process.

The project focuses on:
- Managing adoption workflows
- Centralizing communication
- Improving operational efficiency for mediators

## What I did here

I used AI tools to create a simple landing page that presents the idea and structure of the product.

This was part of exploring how AI can support rapid prototyping and product visualization.

This project reflects my interest in:
- digital product delivery
- structured workflows
- practical use of AI in everyday work
